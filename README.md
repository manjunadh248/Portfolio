# 🚀 Simanjunadha Venkata Pradeep B — Smart Portfolio

A modern, AI-powered personal portfolio website built with 
React, TypeScript, GSAP animations, and Gemini AI chatbot.

## ✨ Features

- 🤖 AI-powered chatbot (Gemini 2.5 Flash) that answers questions 
  about my skills, projects, and experience
- 🎨 Smooth GSAP scroll animations throughout
- 🌐 Sections: Hero, About, Work Experience, Projects, 
  Achievements, Certificates, Contact
- 📱 Fully responsive across all devices
- 💌 Working contact form via Formspree
- ⚡ Spline 3D robot in hero section
- 📄 Resume preview modal with download option

## 🛠️ Tech Stack

- **Frontend:** React 18, TypeScript, Tailwind CSS
- **Animations:** GSAP + ScrollTrigger
- **UI Components:** shadcn/ui
- **AI Chatbot:** Google Gemini 2.5 Flash API
- **Contact Form:** Formspree
- **3D:** Spline
- **Build Tool:** Vite
- **Deployment:** Netlify

## 🚀 Getting Started

### Prerequisites
- Node.js 18+
- npm or bun

### Installation

1. Clone the repository
```bash
git clone https://github.com/manjunadh248/Smart-Portfolio.git
cd Smart-Portfolio
```

2. Install dependencies
```bash
npm install
```

3. Set up environment variables
```bash
cp .env.example .env
# Add your Gemini API key to .env
```

4. Start development server
```bash
npm run dev
```

5. Build for production
```bash
npm run build
```

## 🌐 Deployment

Deployed on Netlify. Environment variables required:
- `VITE_GEMINI_API_KEY` — Google Gemini API key
